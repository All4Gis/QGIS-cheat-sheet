# QGIS cheat sheet

Cheat sheet for PyQgis


Processing algorithms 
---

__Get algorithms list__

	import processing
	processing.alglist()

__Get algorithms Help__

Random selection

	processing.alghelp('qgis:randomselection')


__Get algorithms Options__

Random selection

	processing.algoptions('qgis:randomselection')
